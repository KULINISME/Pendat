# Data Understanding

## Pengertian Data Understanding

Data Understanding ialah ahapan krusial dalam metodologi data science (CRISP-DM) untuk mengenali, mengumpulkan, dan mengeksplorasi data awal guna memahami struktur, kualitas, dan potensi masalahnya (seperti missing values atau outlier). Tahap ini memastikan data relevan dengan tujuan bisnis, mencakup pengumpulan data, deskripsi, eksplorasi (EDA), dan verifikasi kualitas data sebelum pemodelan. 
Berikut adalah setiap langkah dalam Data Understanding.
1. Data Collection
2. Data Description
3. Data Exploration
4. Data Quality Verification

## Pentingnya Memahami Data

Pentingnya Memahami Data adalah agar proses analisis dapat dilakukan secara tepat, akurat, dan sesuai tujuan. Tanpa memahami data, seseorang berisiko salah dalam memilih metode, salah menginterpretasikan hasil, atau bahkan menarik kesimpulan yang keliru.
<br>
Memahami data membantu mengetahui struktur dataset, jenis variabel, distribusi nilai, serta hubungan antarvariabel. Selain itu, pemahaman yang baik memungkinkan kita mendeteksi masalah seperti missing value, duplikasi, outlier, atau kesalahan input sejak awal.
Dengan memahami data, proses pengolahan dan pemodelan menjadi lebih terarah. Keputusan yang diambil pun menjadi lebih valid karena didasarkan pada kondisi data yang benar-benar dipahami, bukan sekadar asumsi.
